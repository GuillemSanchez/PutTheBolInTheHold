# PutTheBolInTheHold

## Controls: 

W - Accelerate
A - Turn Left
S - Break Softly / Go backwards.
D - Move Right
SPACE - Break
Q to look avobe, (beacon). 
R to Re-Position. 

### DEBUG:
F1 - Debug Physic Bodys
F5 - Restart Level 
F9 - Free Camera 

### Free Camera Movement: 

ARROWS - Move Camera
LEFT SHIFT - Lift Camera
LEFT CONTROL - Descend Camera
RIGHT SHIFT - Sprint
